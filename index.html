Summary

list vs tuple

list: mutable vs immutable

1) list + tuple: ordered 순서가 정해져있다 - 내용물이 자료구조에 들어간 순서대루

2) 멤버쉽 검색이 느리다

set vs dictionary

set: key value 가 필요 없고 key 만 필요 할때

dict: key, value가 필요 할때

1) set + dictionary: unordered

2) 검색이 빠르다
