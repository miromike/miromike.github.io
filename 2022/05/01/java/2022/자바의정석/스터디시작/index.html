자바의정석 스터디 7주 시작 하다

구하고 구하고 있는 스터디…
구한곳은 늦게 시작 해서

오키 통해 구했다. 열심히 해보자. 스터디 참여 하시는 분들 블로그 보니 다들 유 경험자 잘 하시는 분들이 많다. 이 스터디 끝나면 이어서 계속 하고 좋은 인연되었으면 좋겠다. 노땅임을 알면 쫓겨나려나…

진도 및 기록을 위해 스터디 모집글을 남겨 본다.

자바의 정석 온라인 스터디 하실분 모십니다. (객체지향부터, 총 7주동안)
자바의 정석 온라인 스터디 하실분 모십니다.

발표 없고, 취준생/실무자이신분들 제한 없습니다.

  • 자바의 정석

  1. 스터디 목표 : 자바의 정석(유튜브 강의) 1회독

  2. 공부 범위

  • 1주차 : 챕터 6 - 객체지향 개념1

  • 2주차 : 챕터 7 - 객체지향 개념2

  • 3주차 : 챕터 8,9,10 - 예외 처리~사용시 유용한 클래스

  • 4주차 : 챕터 11 - Collections Framework

  • 5주차 : 챕터 12 - Generics, Enum, Annotation

  • 6주차 : 챕터 13 - Thread

  • 7주차 : 챕터 14 - Lambda, Stream

  1. 시간 : 주 4회 (매주 월목, 오후 8시10시)

  2. 기간 : 5월 2일 ~ 6월 18일 (총 7주)

  3. 진행 방식

  • 매일 시작 전 공부 시작하는 챕터, 종료 후 완료한 챕터 공유

  • 공부한 내용을 당일 또는 주말에 1챕터 내용 요약 및 정리하여 포스팅 후 링크 공유

    • 주 1회 이상 1챕터 내용을 요약 및 정리한 포스팅 필수

  • 게더타운 또는 디스코드에 모여서 화면공유 킨 후 각자 자바의 정석 인강 시청

  • 1주간 1챕터 분량 못채울 경우 주말에라도 진행

  1. 기타

  • 블로그 없으실 경우 블로그 제작(velog, notion, github, tistory 등) 해두셔야 합니다.

  • 책 있으시면 책 보고 하셔도 되고, 없으셔도 무관 합니다.

  • 챕터별로 필요한 시간이 달라서 시간이 남는 경우 포스팅 또는 복습, 부족할 경우 시간 합의 후 주말에 추가적으로 진행 합니다.

  • 알고있는 내용이라고 대충 하거나, 중도포기 하실분은 신청하지 말아주세요.

  • 인원은 저 포함 최소 5명 ~ 최대 8명으로 생각중입니다.
