Jar 가져와서 쓰는 거 어떻게?

외부 모듈을 쓰는데
Maven, Gradle 사용 안 쓰고 해보려고 한다.

계속 에러 발생 해서
기본적인 jar 파일 공부가 필요하다가 느껴졌다.

User.java 를 Jar 파일로 만들고
Main.java 에서 User.jar 사용 하는 예제를 돌려보고 싶다.

User.java

‘’’
public class User {
private String name;
int idx;
}

‘’’

Main.java

‘’’
public class Main {

public static void main(String[] args) {

    User user = new User();
    System.out.println(user);
}

}
‘’’

2개 파일은 각자 다른 폴더, User와 Main에 두었다.
~/User/User.java
~/Main/Main.java

User 컴파일 및 Jar 파일 생성

javac User.java
jar cf User.jar User.class

Jar파일을 Main 폴더로 복사 하고 Main 컴파일

javac Main.java
User symbol 을 못 찾겠다고 컴파일 에러

javac -cp User.jar Main.java
컴파일 성공

실행

java Main
Exception in thread “main” java.lang.NoClassDefFoundError: User
at Main.main (Main.java:5)
User class 못 찾는 런타임 에러로 보인다.

java -cp User.jar Main
Error: Could not find or load main class Main

여기서 막혔다..
